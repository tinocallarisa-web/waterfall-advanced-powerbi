Quick Start

  1. Add the visual to your Power BI report from Microsoft AppSource.
  2. Drag a Category column (bar labels) into the Category role.
  3. Drag a numeric Value measure into the Value role.
  4. Optionally connect Bar Type (column with values: total, subtotal, delta) and Sort Order (numeric column) for full control.

Data Roles

RoleTypeRequiredDescription
CategoryColumn (text)✅Bar labels
ValueMeasure✅Bar values (positive = gain, negative = loss)
TargetMeasure❌Target value shown as dashed line on totals
Bar TypeMeasure (text)❌Values: total, subtotal, delta. Auto-detected if not connected.
Sort OrderMeasure (numeric)❌Controls bar order. Use a numeric column set to "Do not summarize".

Bar Type Auto-detection

If you don't connect the Bar Type role, the visual automatically treats the first and last rows as totals and all middle rows as deltas. For full control (including subtotals), connect a text column with values total, subtotal, or delta.

AI Narratives Setup (Finance)

  1. Get an API key from console.anthropic.com (Claude) or platform.openai.com (GPT).
  2. In Power BI, open the Format pane → AI Narratives (Finance).
  3. Select your AI provider, paste your API key, and choose the narrative language.
  4. Click the ✦ Generate narrative button in the visual.

Note: AI narratives require Power BI Service (online). The feature is not available in Power BI Desktop due to browser security restrictions on external API calls.

FAQ

Bars appear in the wrong order.
Connect a numeric column to the Sort Order role. In the data model, set that column to "Do not summarize".

Totals and subtotals show as 0.
Make sure the Bar Type column contains exactly total, subtotal, or delta (lowercase). Check for trailing spaces.

I upgraded to Pro but bars are still limited.
Sign out of Power BI and sign back in to refresh your license status.

AI narrative shows "Failed to fetch".
AI narratives only work in Power BI Service (online), not in Power BI Desktop. Upload your report to app.powerbi.com.
